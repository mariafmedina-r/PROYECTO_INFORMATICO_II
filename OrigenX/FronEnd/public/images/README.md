# /public/images

Imágenes estáticas accesibles por URL directa (sin importar en el código).

## Uso

```html
<img src="/images/hero-cafe.jpg" alt="..." />
```

## Subcarpetas sugeridas

- `heroes/`     → imágenes de fondo para secciones hero
- `regions/`    → fotos de regiones cafeteras (Huila, Antioquia, Nariño...)
- `producers/`  → fotos de productores o fincas
- `products/`   → imágenes de productos de ejemplo
- `icons/`      → íconos SVG o PNG estáticos

## Notas

- Formatos recomendados: WebP (preferido), JPG, PNG, SVG
- Tamaño máximo recomendado: 500 KB por imagen
- Las imágenes aquí son servidas tal cual por Vite, sin procesamiento
