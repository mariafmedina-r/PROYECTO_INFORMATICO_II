# Entrega 2

## Descripción
Análisis comparativo de soluciones para el sistema Marketplace Cafetero.

## 📊 Contenido
- Documento de análisis y justificación
- Matriz de comparación de alternativas

## ✅ Resultado
Se seleccionó la alternativa C (Arquitectura híbrida), debido a su equilibrio entre escalabilidad, costos, cumplimiento de requerimientos y flexibilidad.

## 🧠 Conclusión
Esta arquitectura permite integrar frontend y backend propio con servicios externos para pagos y logística, facilitando el crecimiento del sistema.
