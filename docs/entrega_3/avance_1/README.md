# Entrega 3 - Avance 1

## Descripción

Este directorio contiene el primer avance de la Entrega 3 del proyecto **Café Origen**, enfocado en la definición del MVP, planificación ágil y organización del equipo.

## Contenido

- Documento principal del avance en formato Word
- Documento en formato PDF
- Archivo de planificación de sprints
- Evidencia de gestión del Sprint 1

##  Objetivo del avance

Definir el Producto Mínimo Viable (MVP) del sistema, estableciendo:

- Funcionalidades principales del sistema
- Backlog del producto
- Planificación de sprints
- Organización del equipo de trabajo
- Metodología ágil utilizada

El MVP busca validar la interacción básica entre productores y consumidores dentro del sistema, incluyendo registro, gestión de productos y proceso de compra. 

## Herramientas utilizadas

- GitHub (control de versiones)
- GitHub Projects (gestión de tareas)
- Postman (pruebas de API)
- Figma (diseño)
- Vercel / Render (despliegue)
